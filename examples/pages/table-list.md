# Table List Page Template

Use this template for searchable, filterable collections.

Required regions:
- page header with one primary action
- filter toolbar
- table with aligned headers, sort affordance, row actions, and selected state
- pagination
- empty, loading, and error states

Design notes:
- filters should stay compact and predictable
- do not wrap table actions into decorative cards
- preserve row density and readable column alignment
- show destructive actions with clear confirmation
