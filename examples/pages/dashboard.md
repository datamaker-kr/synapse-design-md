# Dashboard Page Template

Use this template for operational overview pages.

Required regions:
- app shell navigation
- page title and compact summary actions
- metric row with stable card dimensions
- primary data region
- secondary activity or status region
- empty, loading, and error states

Design notes:
- keep the first viewport information-dense
- use accent color only for primary action, selected state, or critical link
- avoid decorative hero sections
- use `title-sm` inside metric cards and `body-md` for values and descriptions
